## 使用安排

- `FolderManifest 1.jpg` 和 `FolderManifest 2.jpg` 用于 Mac App Store 产品页展示。
- 上述两张展示图同时嵌入仓库根目录 `README.md` 的“软件界面”部分，作为 GitHub 首页的功能展示。
- `screenshot.png` 作为软件原始界面截图和设计存档，不作为当前 GitHub 首页展示图。
- 第二张展示图中包含 `/Users/mac/Downloads/softs/fileDemo/` 示例路径；正式提交 App Store 前需要再次确认该路径是否需要匿名化。

## 规格检查

- 两张 App Store 展示图均为 2560×1600，画面比例为 16:10。
- 实际软件截图为 1920×1113，不与前两张作为同一组商店展示图使用。
- 具体数量、像素尺寸、格式和其他要求，以 App Store Connect 提交页面的当前提示为准。

## 画面内容检查

- `FolderManifest 1.jpg` 使用中文界面，没有展示用户选择的真实文件夹路径。
- `FolderManifest 2.jpg` 使用演示目录 `fileDemo`，展示的文件名应在提交前再次确认不含隐私或受限信息。
- 展示图中的搜索、高亮、Tree、排序和 Finder 定位功能必须在最终提交版中可正常使用。
- 截图不应出现真实个人姓名、客户资料、账号、通知或其他私密信息。
