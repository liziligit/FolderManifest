# Mac App Store 截图清单

此目录用于存放 FolderManifest 正式提交版本的商店截图。截图必须来自准备上传的实际构建，并与 App Store 产品页描述保持一致。

## 规格

- 使用 PNG 或 JPG；
- 建议准备 3～5 张清晰截图；
- 所有截图使用一致的窗口大小、画面比例和视觉风格；
- 具体数量、像素尺寸和格式要求以 App Store Connect 提交页面的当前提示为准。

## 建议文件名与内容

1. `01-tree-overview.png`：完整主界面，展示清晰的文件夹 Tree 和统计信息。
2. `02-live-scanning.png`：扫描大型示例目录，展示动态扫描状态和已发现项目数。
3. `03-regex-search.png`：展示搜索关键词高亮、“上一个”“下一个”和匹配计数。
4. `04-sorting-options.png`：展示名称、类型、修改时间、大小和文件夹优先等排序能力。
5. `05-copy-and-export.png`：展示复制完整 Tree 与导出 TXT 的操作入口。

## 示例目录要求

- 使用专门为截图准备的虚构目录和文件名；
- 目录层级应足以体现 Tree 的价值，但不要密集到无法阅读；
- 可以使用“项目资料”“设计素材”“会议记录”等通用名称；
- 不得出现真实个人姓名、客户名称、账号、内部项目、学校资料或私人文件路径。

## 拍摄检查

- 使用准备提交的 Release 版本；
- App 图标、名称、版本和界面语言均为最终状态；
- 窗口完整，没有被其他窗口、通知或鼠标提示遮挡；
- Tree 文字清晰，路径没有无意义截断；
- 扫描、搜索、高亮、排序和导出等截图功能均真实可用；
- 不显示桌面文件、菜单栏隐私信息或其他 App 内容；
- 不对截图进行会夸大或虚构功能的修改；
- 拍摄后逐张检查文件尺寸和 App Store Connect 接受情况。
